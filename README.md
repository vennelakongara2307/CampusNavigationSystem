\# Campus Navigation System



\## Overview



The Campus Navigation System is a web-based application designed to help students, faculty, and visitors easily locate buildings, classrooms, laboratories, libraries, offices, and other facilities within a college campus.



\## Features



\* Search for campus locations

\* View building and facility information

\* Easy-to-use interface

\* Database-driven location management

\* Quick navigation assistance for students and visitors



\## Technologies Used



\* Python

\* Flask

\* HTML

\* CSS

\* JavaScript

\* SQLite



\## Project Structure



```text

campus\_navigation/

├── static/

├── templates/

├── app.py

├── campus.db

├── create\_db.py

└── README.md

```



\## Installation



1\. Clone the repository:



```bash

git clone https://github.com/vennelakongara2307/CampusNavigationSystem.git

```



2\. Navigate to the project folder:



```bash

cd CampusNavigationSystem

```



3\. Install dependencies:



```bash

pip install flask

```



4\. Run the application:



```bash

python app.py

```



5\. Open your browser and visit:

# Campus Navigation System

## Overview

The Campus Navigation System is a web-based application designed to help students, faculty, and visitors easily locate buildings, classrooms, laboratories, libraries, offices, and other facilities within a college campus.

## Features

* Search for campus locations
* View building and facility information
* Easy-to-use interface
* Database-driven location management
* Quick navigation assistance for students and visitors

## Technologies Used

* Python
* Flask
* HTML
* CSS
* JavaScript
* SQLite

```text
campus_navigation/
├── static/
├── templates/
├── app.py
├── campus.db
├── create_db.py
└── README.md
```

1. Clone the repository:

```bash
git clone https://github.com/vennelakongara2307/CampusNavigationSystem.git
```

```bash
cd CampusNavigationSystem
```

```bash
pip install flask
```
